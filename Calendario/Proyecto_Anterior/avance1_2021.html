Avance 1. Buscando problemas para mi Proyecto final 
.

Objetivo:

Analizar problemas que se puedan tratar o resolver mediante algoritmos y con programación.


Forma de trabajo:

Actividad individual


   Instrucciones:
 

      En esta actividad deberás buscar problemas que usarás como base para trabajar y seleccionar tu proyecto final.

  • Buscar dos problemas de tu interés personal. Los problemas deben poder representarse mediante un algoritmo.
  • El proyecto final debe incluir el uso de matrices, arreglos o cadenas de caracteres. Es obligatorio el uso de archivos de texto para leer y/o escribir información.
  • Se evaluará la dificultad de los problemas planteados y de acuerdo a ello será su calificación.  
  • El problema debe tener una descripción breve sobre: 
    • ¿Qué es?
    • ¿Qué se desea?
    • ¿Por qué proceso se resuelve? 
      • .
      Ejemplos de entregables del primer avance "Buscando problemas para mi proyecto final"
    ,
  • Referencias formales de donde se informaron sobre el problema
  • Ten en mente que se espera que puedas hacer un algoritmo de este proceso.
  • Elabora un documento donde presentes tu investigación. Revisa los pros y contras del problema y analiza su dificultad.
  • Algunos ejemplos muy genéricos de problemas son:
    • Realizar juegos.
    • Como convertir unidades numéricas de forma compleja como por ejemplo números arábigos a romanos y viceversa.
    • Hacer simulaciones pequeñas sobre algún proceso estocástico como la evolución.
    • Realizar un programa para el aprendizaje o comprensión de un proceso.
